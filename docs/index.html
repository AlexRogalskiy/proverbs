Description

Styled proverbs_ is a serverless function that generates dynamically styled proverb images based on SVG (Scalable Vector Graphics). For the tech stack, _Styled proverbs using Typescript and serverless function from Vercel stack.

How to use

It's simple, you can copy paste this markdown content, like this one:

![Styled proverbs](https://styled-proverbs.vercel.app/api?backgroundColor=[value]&opacity=[value]&colorPattern=[value]&fontColor=[value]&pattern=[pattern]&language=[value]&width=[width]&height=[height]&keywords=[keywords])

There are several options you can use from the list:

Options Description Type Example Query Params
[Background Color] Background color for the proverb image Hex color code %23ffffff ?backgroundColor=[value]
[Opacity Pattern] Background opacity of the pattern Float number 0 - 1 &opacity=[value]
[Color Pattern] Color pattern for the signage Hex color code %231abc9c &colorPattern=[value]
[Font Color] Font color for the proverb text Hex color code %23000000 &fontColor=[value]
[Pattern] Pattern for the background String constant plus &pattern=[value]
[Language] Proverb language String constant swedish &language=[value]
[Width] Proverb image width String 100% &width=[value]
[Height] Proverb image height String 100% &height=[value]
[Keywords] Proverb keywords String/String[] love &keywords=[value]
